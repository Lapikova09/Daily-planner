О проекте (12.2024-01.2025):

Daily Planner — это минималистичный персональный органайзер, разработанный для эффективного управления повседневными задачами и отслеживания прогресса в борьбе с вредными привычками. 

Главная особенность проекта: он построен исключительно на чистом Node.js (Native HTTP модуль) без использования сторонних бэкенд-фреймворков и внешних зависимостей (npm-пакетов).

 Ключевые возможности:
- Управление делами (Tasks): Создание, редактирование, удаление и автоматическая сортировка по дедлайнам.
- Борьба с привычками (Addictions): Трекер «чистых» дней с интерактивным опросником и возможностью мгновенного сброса счётчика при срыве.
- Дневник заметок: Ведение локальной истории записей и рефлексии для каждой привычки.
- Визуализация прогресса: Цветовая индикация статусов выполнения на фронтенде.

Функциональные возможности

 Управление делами
- Просмотр и фильтрация: Отображение списка задач с автосортировкой по ближайшему дедлайну.
- CRUD операции: Полный цикл управления задачами (Добавление, Редактирование, Удаление).
- Интерактивный статус: Быстрое переключение состояния выполнения (красный/зелёный маркер) через клиентский CSS.

 Трекер привычек
- Счётчик «чистых» дней: Автоматический подсчёт времени без вредной привычки.
- Кнопка «Я сорвался»: Мгновенное обнуление счётчика дней для фиксации нового старта.
- Дневник: Возможность добавлять текстовые заметки к каждой привычке для отслеживания триггеров и эмоционального состояния.
 Структура проекта

Архитектура приложения построена по классическому принципу разделения обязанностей (Routing -> Controllers -> Models/Data)
